# Moomoo Reports Convertor

## Installation
Download repository
```
$ cd ~/
$ git clone https://github.com/weikunhan/moomoo-reports-convertor.git
```
Create a [Python virtual environment](https://docs.python.org/3/library/venv.html)
```
$ python3 -m venv ~/moomoo-reports-convertor/venv
```
Active a Python virual environment
```
$ source ~/moomoo-reports-convertor/venv/bin/activate
```
Deactivate a Python virtual environment
```
$ deactivate
```
